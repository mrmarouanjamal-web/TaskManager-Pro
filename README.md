# 📋 TaskManager Pro

Python Task Manager application to organize daily tasks and improve productivity.

## 🚀 Features

- Add tasks
- Show tasks
- Delete tasks
- Edit tasks
- Complete tasks
- Search tasks
- Sort by priority
- Deadlines
- Save automatically in tasks.txt

## 🛠 Technologies

- Python
- Lists
- Functions
- File handling
- Loops
- Search
- Sorting

## ▶ Run

```bash
python main.py
```

## Example Menu

```text
1. Add task
2. Show tasks
3. Delete task
4. Edit task
5. Complete task
6. Search task
7. Sort tasks
8. Exit
```
